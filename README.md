## Download project
```bash
git clone https://github.com/is-tech-y24-1/Do-Svyazi.User.git
```

## Restore dependencies
```bash
dotnet restore
```

## Start project
```bash
dotnet run --project .\Source\Infrastructure\Do-Svyazi.User.Web.Api\
```

## Run unit tests
```
dotnet test --no-build
```
